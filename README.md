# Gledhill Metadata

## Linux Quick Install

[![Install AppImage](https://img.shields.io/badge/Linux-Install%20AppImage-2ea44f?style=for-the-badge&logo=linux)](https://raw.githubusercontent.com/Nerd-or-Geek/Gledhill-Metadata/main/linux/install.sh)
[![Uninstall AppImage](https://img.shields.io/badge/Linux-Uninstall%20AppImage-bd2c00?style=for-the-badge&logo=linux)](https://raw.githubusercontent.com/Nerd-or-Geek/Gledhill-Metadata/main/linux/uninstall.sh)

Install from curl:

```bash
curl -fsSL https://raw.githubusercontent.com/Nerd-or-Geek/Gledhill-Metadata/main/linux/install.sh | bash
```

Uninstall from curl:

```bash
curl -fsSL https://raw.githubusercontent.com/Nerd-or-Geek/Gledhill-Metadata/main/linux/uninstall.sh | bash
```

Build tools setup (Linux):

```bash
bash linux/setup_linux_build.sh
```

Build AppImage locally:

```bash
bash linux/build_appimage.sh
```

The Linux helper scripts now live in [linux](linux).

## Features

- **Metadata Editing** — Update XMP/EXIF/IPTC fields across large photo sets
- **Reusable Templates** — Save presets and apply them in one click
- **Batch Rename** — Rename entire folders using pattern rules instantly
- **Local-first** — Your photos never leave your computer
